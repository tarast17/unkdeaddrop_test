# Simuation of VSCode Task Delivery

⚠️ This is a **security simulation project** inspired by threat research.

This repo demonstrates:
- VS Code task execution
- Encoded PowerShell usage
- Developer-focused workflow mimicry

✅ Safe payload:
- Launch calc OR print message

## How to run

1. Open folder in VS Code
2. Press:
   Ctrl + Shift + P → Run Task
3. Select: "Initialize Project"
